客户端Web-API

1 Q&A

Q：什么是API？

A：应用程序接口（API，Application Programming Interface）是基于编程语言构建的结构，使开发人员更容易地创建复杂的功能。它们抽象了复杂的代码，并提供一些简单的接口规则直接使用。

Q：为什么需要API？

A：加速开发，避免造轮子。

Q：JavaScript和API以及库、框架的关系？

A：JavaScript，是一种运行于浏览器或者Node.js中的高级语言。

客户端API，是一种提供高级功能的代码段，可直接调用。

JavaScript库，是包含特定功能的一个或多个JavaScript文件，引入这些文件可以避免重复实现功能。

JavaScript框架，将HTML，CSS，JavaScript技术结合在一起，从头开始创建一个Web应用。

2 都有哪些API

  • 浏览器内置API
    • 操作文档：例如DOM，允许脚本操作HTML和CSS。
    • 从服务器获取数据：例如XMLHttpRequest，Fetch API，允许异步获取数据并局部更新页面。
    • 用于绘制和操作图形：例如canvas，允许网页中绘制图形
    • 音频和视频：例如HTMLMediaElement，Web Audio API
    • 设备 API：检索客户端设备硬件数据，例如Notification，地理API
    • 客户端存储：保存页面之间加载状态，使应用脱机使用，Web Storage API存储简单键值对, IndexedDB API存储更复杂数据
  • 第三方API
    • Twitter
    • Google Maps
    • Twilio
    • 在此找到更多

3 JavaScript中API如何工作

  • 基于对象，通过对象调用API
  • 有可识别入口点，可以根据功能来找到被调用的API对象
  • 可使用事件处理状态变化，例如XMLHttpRequet中，当数据返回时，通过onload异步调用处理函数
  • 有一定安全机制，例如同源策略等
